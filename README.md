# FoodieGo - Online Food Delivery Website

This is an online food delivery website using HTML, CSS and JavaScript with comprehensive user and admin dashboards.

## Features

### Main Website
- Responsive design with modern UI
- Interactive menu with categories
- Shopping cart functionality
- User reviews and testimonials
- Newsletter subscription
- Contact information

### User Dashboard
- **Overview**: Personal statistics, recent orders, spending summary
- **Order Management**: View order history, track current orders, reorder functionality
- **Favorites**: Save and manage favorite food items
- **Profile Management**: Update personal information and profile picture
- **Address Book**: Manage multiple delivery addresses
- **Payment Methods**: Store and manage payment cards
- **Settings**: Notification preferences and privacy controls

### Admin Dashboard
- **Dashboard Overview**: Revenue, orders, users, and product statistics
- **Order Management**: View, edit, and manage all customer orders
- **Product Management**: Add, edit, delete, and toggle product availability
- **User Management**: View and manage customer accounts
- **Analytics**: Revenue trends, popular items, customer growth charts
- **Review Management**: Approve, delete, and manage customer reviews
- **System Settings**: Configure delivery, payment, and general settings

## Getting Started

1. Open `main.html` in your web browser to view the main website
2. Click "User Dashboard" to access the customer dashboard
3. Click "Admin Panel" to access the administrative dashboard
4. All dashboards are fully functional with demo data
